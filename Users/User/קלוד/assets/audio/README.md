# אודיו המשחק

שים כאן קבצי אודיו ועדכן את `ASSETS.audio` ב-`game.js`.

| מפתח ב-ASSETS | שם קובץ מומלץ | תיאור |
|---|---|---|
| `click` | `click.mp3` | צליל קליק על האובייקט הראשי |
| `purchase` | `purchase.mp3` | צליל רכישת בניין/שדרוג |
| `milestone` | `milestone.mp3` | צליל הישג / מיילסטון |
| `ambient` | `ambient.mp3` | מוזיקת רקע (loop) |
| `golden` | `golden.mp3` | צליל הופעת עוגיית זהב |
