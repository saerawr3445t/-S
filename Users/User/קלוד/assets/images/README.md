# תמונות המשחק

שים כאן קבצי תמונות ועדכן את `ASSETS.images` ב-`game.js`.

| מפתח ב-ASSETS | שם קובץ מומלץ | תיאור |
|---|---|---|
| `mainObject` | `main.png` | האובייקט המרכזי לחיצה (עגול 200×200) |
| `background` | `bg.jpg` | תמונת רקע |
| `goldenCookie` | `golden.png` | עוגיית הזהב הזזה (64×64) |
| `buildingIcons.cursor` | `cursor.png` | אייקון סמן (34×34) |
| `buildingIcons.farm` | `farm.png` | אייקון חווה |
| `buildingIcons.mine` | `mine.png` | אייקון מכרה |
| `buildingIcons.factory` | `factory.png` | אייקון מפעל |
| `buildingIcons.bank` | `bank.png` | אייקון בנק |
| `buildingIcons.temple` | `temple.png` | אייקון מקדש |
| `buildingIcons.wizard` | `wizard.png` | אייקון מגדל קוסמים |
| `buildingIcons.shipment` | `shipment.png` | אייקון משלוח |
| `buildingIcons.alchemy` | `alchemy.png` | אייקון מעבדת אלכימיה |
